Privacy Notice for RobloxCrashMonitor
Last Updated: November 2025

Overview
RobloxCrashMonitor (“the Utility”) is a local desktop tool that helps users detect Roblox-related crashes on their own device. This Privacy Notice explains how the Utility operates and what information it does not collect.

No Data Collection
RobloxCrashMonitor does NOT collect, store, upload, or transmit any personal information. This includes, but is not limited to:
• usernames
• IP addresses
• Roblox account information
• gameplay data
• system telemetry
• analytics
• logs sent to any server

All monitoring is performed locally on your device.

Files Read by the Utility
RobloxCrashMonitor only reads:
• Roblox crash dump files
• Roblox log files
• Local folders necessary to detect crash events

These files remain on your device. They are never uploaded or shared.

Optional User Configuration
If you edit the config.json file, the following preferences are stored locally:
• Webhook URL (optional)
• Paths to monitored folders
• Scan interval

No Network Communication
By default, RobloxCrashMonitor:
• Does not access the internet
• Does not communicate with external servers
• Does not include telemetry
• Sends data ONLY if the user provides a webhook URL

Open-Source Transparency
The included source code allows anyone to verify that:
• No hidden connections exist
• No data collection occurs
• No background tracking is performed

User Responsibility
Users are responsible for:
• How they use the Software
• Any modifications they make
• Their system configuration
